GoCV gives programmers who use the Go programming language access to the OpenCV 3 computer vision library.

The GoCV package supports the latest releases of Go and OpenCV v3.4 on Linux, macOS, and Windows. We intend to make the Go language a “first-class” client compatible with the latest developments in the OpenCV ecosystem.

GoCV also supports the Intel Computer Vision SDK using the Photography Vision Library (PVL).

Just looking for the code? Go to https://github.com/hybridgroup/gocv
